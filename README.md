# Презентація Польщі — мобільне веб-додаток

Цей проєкт — адаптивна односторінкова презентація Польщі українською мовою. Вона використовує дизайнерський підхід з градієнтами, скрол-стопами й картами, щоб кожен слайд виглядав як окрема картка на мобільному екрані.

## Структура
- `index.html` — семантичний макет з титульним блоком і десятьма «слайдами», що покривають інформацію про географію, економіку, ресурси, демографію, історію та цікаві факти.
- `styles.css` — оформлення з використанням шрифтів Manrope/Playfair Display, градієнтів, карток, hover-ефектів і плавного скролу.

## Інструкції
1. Відкрий `index.html` у браузері (наприклад, подвійним кліком або через `live-server`), щоби побачити версію, адаптовану під мобільні ширини.
2. Якщо хочеш побачити скролл-сенс, прокручуй сторінку одним пальцем — кожна секція «відрізняється» завдяки `scroll-snap`.
3. Щоб кастомізувати контент, редагуй внутрішні блоки `<section class="slide">` у `index.html`; стилі автоматично прокладуть макет.

## Додатково
- Шрифти завантажуються через Google Fonts для кінцевого вигляду.
- Кольори й фон налаштовані на темну палітру з акцентами синьо-рожевого теплого світла.
